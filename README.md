# 💐 FlowerShopAPI
מערכת לניהול חנות פרחים — מאפשרת לנהל מוצרים, לקוחות והזמנות.

---

## 📋 תיאור הפרויקט
המערכת מספקת ממשק API לניהול מלאי מוצרים (זרי פרחים, עציצים וכו׳), ניהול לקוחות קבועים וטיפול בהזמנות כולל סטטוס.  
הפרויקט נבנה כ־**ASP.NET Core Web API** עם תמיכה ב־**Swagger** לבדיקת הקריאות.

---

## 🧱 ישויות עיקריות
1. **Product** – מייצגת מוצר בחנות (שם, מחיר, כמות במלאי).  
2. **Customer** – מייצגת לקוח קבוע (שם, טלפון, אימייל).  
3. **Order** – מייצגת הזמנה של לקוח (מוצרים, תאריך, סטטוס, סכום כולל).

---

## 🌐 מיפוי Routes לפי תקן REST

### 🪻 Product
| פעולה | HTTP Method | נתיב | תיאור |
|--------|--------------|-------|--------|
| שליפת כל המוצרים | GET | `/api/products` | מחזיר רשימת מוצרים |
| שליפת מוצר לפי מזהה | GET | `/api/products/{id}` | מחזיר מוצר יחיד לפי מזהה |
| הוספת מוצר חדש | POST | `/api/products` | מוסיף מוצר חדש למערכת |
| עדכון מוצר קיים | PUT | `/api/products/{id}` | מעדכן נתוני מוצר קיים |
| מחיקת מוצר | DELETE | `/api/products/{id}` | מוחק מוצר |
| עדכון מלאי בלבד | PUT | `/api/products/{id}/stock` | משנה את כמות המלאי בלבד |

---

### 👩‍💼 Customer
| פעולה | HTTP Method | נתיב | תיאור |
|--------|--------------|-------|--------|
| שליפת כל הלקוחות | GET | `/api/customers` | מחזיר רשימת לקוחות |
| שליפת לקוח לפי מזהה | GET | `/api/customers/{id}` | מחזיר פרטי לקוח יחיד |
| הוספת לקוח חדש | POST | `/api/customers` | מוסיף לקוח חדש למערכת |
| עדכון פרטי לקוח | PUT | `/api/customers/{id}` | מעדכן פרטי לקוח קיים |
| מחיקת לקוח | DELETE | `/api/customers/{id}` | מוחק לקוח מהמערכת |
| שליפת כל ההזמנות של לקוח | GET | `/api/customers/{id}/orders` | מציג את כל ההזמנות של לקוח מסוים |

---

### 🧾 Order
| פעולה | HTTP Method | נתיב | תיאור |
|--------|--------------|-------|--------|
| שליפת כל ההזמנות | GET | `/api/orders` | מחזיר רשימת הזמנות |
| שליפת הזמנה לפי מזהה | GET | `/api/orders/{id}` | מחזיר הזמנה אחת לפי מזהה |
| הוספת הזמנה חדשה | POST | `/api/orders` | מוסיף הזמנה חדשה |
| עדכון הזמנה קיימת | PUT | `/api/orders/{id}` | מעדכן נתונים בהזמנה |
| מחיקת הזמנה | DELETE | `/api/orders/{id}` | מוחק הזמנה מהמערכת |
| עדכון סטטוס הזמנה | PUT | `/api/orders/{id}/status` | משנה סטטוס (לדוגמה: בהכנה / נשלחה / הושלמה) |

## ⚙️ טכנולוגיות
- **ASP.NET Core 8.0**  
- **C#**  
- **Swagger / OpenAPI**  
- **Entity Framework Core** *(אופציונלי)*  

---

## 🚀 הוראות הרצה
1. לפתוח את הפרויקט ב־**Visual Studio**.  
2. להריץ (Ctrl + F5).  
3. להיכנס לכתובת:  
https://localhost:{port}/swagger

4. לבדוק את כל הפעולות דרך Swagger UI.
